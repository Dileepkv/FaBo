## ケース構成

![](./img/raspb2b_00.jpg)

使用ネジはM3-30プラネジ6本です。

重ねる際は方向に注意してください。

## 組み立て方法
1.　(1)の下からネジを図のようにセットします。

![](./img/raspb2b_01.jpg)

ナットでの固定はしません。

2.　手順１でセットしたネジに合わせて、図のようにパーツ(2)、(3)を重ねます。

![](./img/raspb2b_02.jpg)

3.　続けて(4)、(5)、(6)を重ねます。

![](./img/raspb2b_03.jpg)

4.　マイコンボードをセットし、被せるように(7)を重ねます。

![](./img/raspb2b_04.jpg)

![](./img/raspb2b_05.jpg)

注意　細い部分があります。

5.　(8)、(9)、(10)を順番に重ねます。

![](./img/raspb2b_06.jpg)

3パーツとも同じ形です。

6.　(11)を重ねます。

![](./img/raspb2b_07.jpg)

7.　ネジをナットで固定します。ネジの方向は使用環境に応じて変えてください。

ゴムクッションはお好みでお使いください。

ネジが長いと感じる方はニッパーなどでカットしてください。
