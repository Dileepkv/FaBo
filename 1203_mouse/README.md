# 配線について

![](./img/h1.png)

モバイルバッテリーより給電するために配線を行います。

白のUSBケーブルはMotor Shieldのジャックに接続し、黒のUSBケーブルはRaspberryPIのUSBポートにそれぞれ接続します。

![](./img/h2.png)

制御するDCモーターを接続します。

ShieldのMotor1、Motor2のコネクタに対して、それぞれ黄色い線が中央になるようにモーターを接続します。

モーターの線は、Shield側のコネクタ上部の白い部分を押す事で差し込むことができます。
