# #503 OUT/IN Shield for IchigoJam
  
![](./img/503_outin_ichigojam.jpg)
<!--COLORME-->

## Overview
OUT/IN Shield for IchigoJamは、IchigoJamと各種センサーやボタンをケーブルを1本接続するだけで使えるIchigoJam対応シールドです。

IchigoJam本体、およびIchigoJamケースは含まれません。別途、お買い求めください。
