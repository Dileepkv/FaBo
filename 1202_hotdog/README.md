# モーターギヤボックス前準備

ギヤ比は 114.7 : 1　組み立てます。

モーターは６V、六角シャフトは150mmを使用します。

<hr>

# タイヤ前準備

前輪タイヤと後輪タイヤを各２本、計４本準備します。

前輪はゴム枠だけ組み立てます。

後輪はシャフト取り付け治具まで組み立てます。ホイール内部が深いほうが内側になります。

# サーボの組み立て

ギヤを中心（初期位置）にします。

方法

![](./img/type1_02_1.png)

クランプパーツを取り付け、手順１～３のように動かし、中心位置にします。

![](./img/type1_02.png)

手順４でネジで固定します。

向きに注意してください。

使用ネジ（M3-12ｍｍ）２本

# マイコンをマウントするネジの取り付け

![](./img/type1_04.png)

取り付けるマイコンによってネジ穴が違います。ガイドに合わせて取り付けてください。

裏から差し込んで固定します。

使用ネジ（M2.6-15ｍｍ）４本

<hr>

# サーボの取り付け

![](./img/type1_05.png)

上から固定パーツで挟むように取り付けます。

サーボの向きに注意してください。

使用ネジ（M3-12ｍｍ）２本

<hr>

# ステアリングパーツの取り付け

![](./img/type1_06.png)

タイヤを取り付けるパーツを組み立てます。

裏からM3ナットを差し込みます。

（構成部品として完成している場合があります）

2個準備します。

![](./img/type1_07.png)

ステアリングバーを取り付けます。

サーボのクランプパーツの溝にはめるように取り付けます。

![](./img/type1_08.png)

フロントパーツで挟むように固定します。

使用ネジ（M3-25ｍｍ）4本

![](./img/type1_10.png)
