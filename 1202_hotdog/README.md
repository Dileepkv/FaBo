# サーボの組み立て

ギヤを中心（初期位置）にします。

方法

![](./img/type1_02_1.png)

クランプパーツを取り付け、手順１～３のように動かし、中心位置にします。

![](./img/type1_02.png)

手順４でネジで固定します。

向きに注意してください。

# マイコンをマウントするネジの取り付け

取り付けるマイコンによってネジ穴が違います。ガイドに合わせて取り付けてください。

裏から差し込んで固定します。

# サーボの取り付け

![](./img/type1_04.png)

上から固定パーツで挟むように取り付けます。

サーボの向きに注意してください。
