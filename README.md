# FaBo

[FaBo](http://fabo.io/)

FaBo Platfom is consist of controllers and shields and bricks of Arduino/Raspberry Pi/IchigoJam/etc.

### Shileds
- BLE
- RS-232C
- DC Motor
- Stepping Motor

### Bricks
- Angle
- Bitton
- Humidity
- LCD
- Light
- Temperlature
- Tilt
- Touch
- UV
- Vibrator

- RTC
- Temperlature
- LCD

### Board

- LPC
